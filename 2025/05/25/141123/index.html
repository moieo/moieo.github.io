Arch Linux UEFI 安装指南：Btrfs 子卷 + GNOME 桌面环境

从零开始配置现代化 Arch Linux 系统（含 Timeshift 快照支持）

2025年05月25日
阅读时长: 4 分钟

声明 原作者信息

  • 哔哩哔哩主页：https://space.bilibili.com/2017026819
  • QQ号：2037566388

🐧 Arch Linux 安装指南 (UEFI专用版) - 完整版

温馨提示：本教程仅适用于UEFI设备哦~ (包括虚拟机模拟UEFI或真机UEFI)，传统BIOS(Legacy)设备的小伙伴请寻找其他教程呢 (｡•́︿•̀｡)

🎒 准备开始

需要准备的东西

  • 最新版的archlinux iso镜像文件
  • 一个U盘(如果你要在实体机安装的话)

重要注意事项

✨ 安全小贴士 ✨

  • 一定要备份好所有重要数据哦！
  • 操作有风险，请谨慎行事~
  • 如果是第一次尝试，强烈建议先在虚拟机里练习一下呢 (｡･ω･｡)

🖥️ 启动Live环境

启动方式选择

  • 使用虚拟机：直接加载下载好的ISO镜像启动
  • 使用实体机：需要用工具把ISO写入U盘后从U盘启动

💾 磁盘分区与配置

使用cfdisk分区工具

1

cfdisk /dev/sda

分区步骤详解

  • 初始化磁盘格式：选择 [gpt] 选项
  • 创建EFI系统分区：
    • 大小设置为500M就够用啦
    • 类型要选 EFI System 哦
  • 创建根分区：
    • 使用剩下的所有空间
    • 类型选择 Linux root (x86-64)

分区表示例

格式化分区操作

1
2
3
4
5

# 格式化EFI分区为FAT32格式
mkfs.fat -F32 -n "ARCH-EFI" /dev/sda1

# 格式化根分区为Btrfs格式
mkfs.btrfs -L "arch-root" /dev/sda2

Btrfs子卷设置

1
2
3
4
5
6
7
8
9

# 先挂载根分区
mount /dev/sda2 /mnt

# 创建必要的子卷
btrfs subvolume create /mnt/@            # 主系统子卷
btrfs subvolume create /mnt/@timeshift   # 备份子卷

# 临时卸载
umount /mnt

智能挂载选项

根据你的硬件配置选择合适的挂载参数：

设备类型推荐挂载命令
老款SSD (2016年前CPU)mount -o subvol=@,compress=zstd:2,noatime,ssd /dev/sda2 /mnt
新款SSD (2017年后CPU)mount -o subvol=@,compress=zstd:3,noatime,ssd /dev/sda2 /mnt
虚拟机(使用SSD存储)mount -o subvol=@,compress=zstd:1,noatime,ssd /dev/sda2 /mnt
机械硬盘(HDD)mount -o subvol=@,compress=lzo,noatime /dev/sda2 /mnt
不确定配置mount -o subvol=@,compress=auto,noatime /dev/sda2 /mnt

创建挂载点结构

1
2
3
4
5

# 创建必要的目录结构
mkdir -p /mnt/{boot/efi,timeshift}

# 挂载EFI分区
mount /dev/sda1 /mnt/boot/efi

📦 安装基础系统

让pacman更可爱

1

nano /etc/pacman.conf

找到并取消注释 Color，然后在下面添加 ILoveCandy，保存退出~

使用国内镜像加速

1
2
3
4
5

# 设置清华镜像源
echo 'Server = https://mirrors.tuna.tsinghua.edu.cn/$repo/os/$arch' > /etc/pacman.d/mirrorlist

# 刷新软件包数据库
pacman -Syy

安装必备软件包

1
2
3

htop cpupower nano dhcpcd net-tools networkmanager iw wpa_supplicant \
bluez bluez-utils blueman grub efibootmgr sudo doas snapper timeshift fastfetch

自动生成fstab

1

进入新系统环境

1

arch-chroot /mnt
